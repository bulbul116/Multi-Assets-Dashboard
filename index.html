Web3 Multi-Asset Viewer

Generate a normalized, chain-agnostic portfolio export for a single wallet.

An API key is required to fetch ERC-20 balances. Without one, EVM chains will be marked as inactive alongside non-EVM chains in the export.

Export JSON

{
  "wallet": "",
  "scan_timestamp": "",
  "total_value_usd": 0,
  "chains": [],
  "assets": [],
  "nfts": []
}
